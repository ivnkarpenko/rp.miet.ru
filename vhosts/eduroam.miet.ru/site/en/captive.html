Eduroam (education roaming) is the secure, world-wide roaming access service developed for the international research and education community. Eduroam allows students, researchers and staff from participating institutions to obtain Internet connectivity across campus and when visiting other participating institutions by simply opening their laptop.

How to connect

Automatic configuration

Configuration Assistant Tool avaliable for all popular platforms (Windows Vista/7/8, Mac OS, iOS, Linux).
You can obtain Configuration Assistant Tool for your platform at https://cat.eduroam.org/?idp=1042.
While running the Configuration Assistant Tool you must enter login and password in the appropriate text areas.
Login format must be like login@miet.ru (ex., u211111@miet.ru or t1000000@miet.ru).

If you have not changed the default password, you can't use EDUROAM Wi-Fi

Manual configuration

If Configuration Assistant Tool avaliable for your platform you shouldn't setup eduroam connection manualy.

Settings for manual configuration:

  • SSID (network name): eduroam
  • Login: your_login@miet.ru
  • Password: your_password
  • Тип сетевой аутентификации: WPA2-enterprise
  • Encryption method: AES
  • Тип протокола проверки подлинности – PEAP (or TTLS)
  • Метод проверки подлинности - MS-CHAPv2

For some platforms you must download and install MIET root certification authority.
